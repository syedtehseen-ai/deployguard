# DeployGuard

DeployGuard is a DevOps Micro-SaaS project designed to analyze Kubernetes deployments and detect configuration risks.

This project is being built through a structured 16-week engineering mission to strengthen cloud architecture, Kubernetes, and DevOps skills.

Current Version: v0.1
Stack:
- FastAPI
- Docker
- Kubernetes (kind)
- PostgreSQL
- ConfigMap / Secret
- PersistentVolume
